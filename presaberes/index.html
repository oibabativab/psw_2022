principales etiquetas HTML

Etiqueta Descripción
<html> Encapsula todo el contenido de una página HTML, también se denomina elemento raíz
<head> Se utiliza como contenedor de todos los elementos que se desean cargar en la página HTML y no es visible a usuarios a excepción del título.
<meta> Contiene los metadatos de la página como el contenido, el idioma del contenido, etc
<title> Corresponde al título del documento
<body> Encapsula el contenido visible al usuario, solo puede existir uno en la página
<div> Utilizado para agrupar contenidos y crear secciones.
<img> Contiene imágenes visibles al usuario.
<h1>, <h2>, …, <h5>, <h6> Encabezados de distinta jerarquía, 1 es la mayor y 6 la menor; son utilizados como títulos dentro de los documentos HTML
<p> Párrafo: se utiliza para presentar al usuario cadenas de texto extensas.
<ul> Crea una lista no ordenada (ul por sus siglas en inglés: unordered list), requiere también el uso de la etiqueta li para cada uno de los elementos que se desean incluir en ella
<a> Permite crear enlaces (links) a otras páginas de internet, páginas dentro del mismo sitio o archivos, correos o cualquier URL

Estilos CSS

Estilo Descripción Sintaxis Referencia
Colors Permite especificar el color a utilizar en el texto dentro del elemento. <color> https://developer.mozilla.org/es/docs/Web/CSS/color
Backgrounds Permite establecer valores individuales a multiples caracteristicas del fondo del elemento, dentro de estas se encuentran: attachment: para establecer si el fondo es fijo o se desplaza con el elemento clip: Señala si el fondo se extiende por debajo de su borde Color: image: origin position repeat size: [ <bg-layer> , ]* <final-bg-layer> https://developer.mozilla.org/es/docs/Web/CSS/background
Borders Permite definir todos los atributos de bordes indicando color, estilo y grosor. <line-width> || <line-style> || <color> https://developer.mozilla.org/es/docs/Web/CSS/border
Margins Asigna todas las propiedades asignables a los margenes de los cuatro lados [ <length> | <percentage> | auto ]{1, 4} https://developer.mozilla.org/es/docs/Web/CSS/margin
Text Son múltipleas las etiquetas que pueden utilizarse para aplicar estilos a textos permitiendo modificar la alineación, el uso de características como subrayado, énfasis o la orientación. En este caso presentamos la sintaxis y el link a text-orientation mixed | upright | sideways https://developer.mozilla.org/es/docs/Web/CSS/text-orientation
Fonts Permite la asignación de valores a las propiedades de estilo, el tamaño de la fuente, el tipo de fuente, etc font: [ <font-style> || <font-variant> || <font-weight> ]? <font-size> [ / <line-height> ]? <font-family> https://developer.mozilla.org/es/docs/Web/CSS/font
Height Especifica el valor de la altura del contenido en un elemento auto | <length> | <percentage> | min-content | max-content | fit-content | fit-content(<length-percentage>) where <length-percentage> = <length> | <percentage> https://developer.mozilla.org/es/docs/Web/CSS/height
Width Asigna el ancho del area de contenido de un elemento auto | <length> | <percentage> | min-content | max-content | fit-content | fit-content(<length-percentage>) where <length-percentage> = <length> | <percentage> https://developer.mozilla.org/es/docs/Web/CSS/width
Links Permite la asignación de estilos a etiquetas de links <a> que puede haber sido visitados, no visitados o que aplica sin importar si fue visitado o no :link https://developer.mozilla.org/es/docs/Web/CSS/:link
List Proporciona una forma de asignar los estilos a listas en tres parámetros diferentes: tipo, imagen y posición. list-style: [ list-style-type || list-style-position || list-style-image ] | inherit https://developer.mozilla.org/es/docs/Web/CSS/list-style
Tables CSS permite aprovechar las etiquetas que conforman el modelo de tablas de HTML para la definición de estilos, revisaremos aquí la etiqueta table-layout que permite indicar el algoritmo a utilizar por el navegador en la distribución de celdas, filas y columnas dentro de la página. auto | fixed https://developer.mozilla.org/en-US/docs/Web/CSS/table-layout
Position Señala como se posiciona un elemento HTML dentro del documento. static | relative | absolute | sticky | fixed https://developer.mozilla.org/es/docs/Web/CSS/position
Overflow Traducida como EXCEDENTE, este estilo permite indicar al navegador si el contenido dentro de un elemento HTML debe recortarse, mostrar barras de desplazamiento o mostrarlo a nivel de bloque [ visible | hidden | clip | scroll | auto ]{1,2} https://developer.mozilla.org/es/docs/Web/CSS/overflow
Float Permite ubicar un elemento en alguno de los lados (izquierdo o derecho) del contenedor en que se encuentra. left | right | none | inline-start | inline-end https://developer.mozilla.org/es/docs/Web/CSS/float
